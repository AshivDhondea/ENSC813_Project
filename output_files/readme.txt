This directory contains results files produced by the scripts developed in this project.

The files are named according to the script which created them.

Figures are either in *.pdf or in *.png form.

Excel files (*.xlsx) and *.tex files contain numerical results.

*.tex files were preferred over *.txt or *.xlsx files since they can be readily included in the LaTeX report.
